# Pokemon 英语学习 App - 图片资源计划

## 目标
用 MiniMax image-01 生成精美图片资源，每天100张额度，需精打细算

## 图片清单（按优先级）

### 优先级1：宝可梦卡片图（PetDetailPage用）
需要：25张（当前有25个Pokemon）
风格：Pokemon TCG风格的闪卡，圆角，有光泽，有Pokemon中英文名+属性图标

Prompt公式：
```
A Pokemon card, holographic foil treatment, centered [Pokemon name] (chibi style), 
Chinese name [中文名] below, card has colored border matching Pokemon type,
soft studio lighting, collectible card game aesthetic, white background,
high detail, vibrant colors, 1:1 aspect ratio
```

### 优先级2：单词配图（Quiz题目用）
需要：120张（当前词汇量）
风格：简单可爱的插图，适合7岁儿童，单词物品居中，简洁背景

Prompt公式：
```
Simple children's book illustration, a cute [物品], 
soft pastel colors, centered composition, white background,
minimal detail, clear outline, adorable style, flat design,
1:1 aspect ratio
```

### 优先级3：界面装饰图
- 学习进度页背景图
- 成就徽章（铜/银/金）
- 空状态插图
